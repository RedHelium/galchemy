# galchemy

[![Package Version](https://img.shields.io/hexpm/v/galchemy)](https://hex.pm/packages/galchemy)
[![Hex Docs](https://img.shields.io/badge/hex-docs-ffaff3)](https://hexdocs.pm/galchemy/)

`galchemy` is a small SQL query builder core for Gleam, focused on:

- Immutable query builders
- Explicit SQL AST
- Predictable SQL compilation with parameters
- PostgreSQL-first integration via `pog`

## Status

This project is currently in active MVP development.



## Development

```sh
gleam check
gleam test
gleam run
```
